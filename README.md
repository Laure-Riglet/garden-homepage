# _agarden_ homepage

## Project Description

This is a project from [Frontend Mentor](https://www.frontendmentor.io/challenges/room-homepage-BtdBY_ENq) to test Vue.js.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
